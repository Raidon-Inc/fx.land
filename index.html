Solution?

BOX is a form factor for personal servers

  • Zero-Config

    Using libp2p, it creates p2p connections behind NATs without any port forwarding, config or static IP

  • Application server

    Using IPFS and IPLD it offers both file and database functionality, with easy-to-use APIs for developers to develop on them.

  • Resiliance and security

    It offers blokchain-level encryption and multiple boxes can easily be clustered and be syncronized.

How it works?

Self-Owned, Secure, Zero-config, and Modular BOX

  • Stationary

    User simply connects BOX to the internet. Multiple boxes can be placed in different physical locations.

  • Zero-config and Secure

    They use BOX smart application to set it up in a few easy clicks, and create a secure and encrypted app server.

  • Modular and Composable

    Users can use differnet distributed applications, such as Photos, directly hosted on their BOX or customize BOX modules.

Is BOX a home-cloud?

Yes and much more!

DApp server

It is designed for distributed applications so unlike traditional home-clouds, no static IP or domain is needed to access your applications outside your home.

Developer friendly

By providing easy-to-use APIs, DApp developers can build applications for Web3.0 without a prolonged learning curve having access to database and files.

Self-sustained

Users can set it to "mine cryptocurrency when idle" mode, and it pays for the DApps by itself, i.e. electricity!

Experience

  • 01. Order

    We ship the user a "BOX" consisting of one right brain, hub, base adn one storage with Photos (like Google photos) and Drive (like onedrive) applications pre-installed. Users can order additional modules as well.

  • 02. Install

    User connects it to insternet and installs mobile application to set it up. App automatically detects that "BOX" and creates an encryption key, IPFS daemon, cluster and database. If user has more than one "BOX", they all get syncronized during initialization.

  • 03. Use

    User can start manual uploading and auto-syncronizing photos and files to their "BOX" using the app from any iOS, Android or Windows device, and share with others who might not even have a "BOX". No IP or domain is needed to use or share.

  • 04. Customize

    User can setup more than one "BOX" in home or office and they all get syncronized. Moreover, multiple users can securely share one "BOX" together, without being abel to see each other data. Also htey can install additional DApps from any developer as they become available.
